## 快递代拿系统 SSM 版

>基于 SpringBoot 全新开发的快递代拿系统正在紧张开发中，相关技术栈全面升级，敬请期待！
>
>2019.04.24

### 使用技术

采用 Spring + SpringMVC + MyBatisPlus，连接池采用 Druid，安全框架使用 Shiro，前端采用 Bootstrap + layer 实现。

### 运行环境

- 集成开发环境：IntelliJ IDEA

- 项目构建工具：Maven

- 数据库：Mysql

- JDK版本：jdk1.8

- Tomcat版本：Tomcat8
